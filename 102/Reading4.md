1. ___What is HTML and why do we use it?___
   
   HTML is a markup language consisting of a series of “elements” used to wrap (or enclose) text content to define its structure and cause it to behave in a certain way. HTML can be used to create text or image links to other web pages, embed images or videos, create data tables, and so on. 
   
1. ___What are the 3 main parts of an HTML element?___
   
   * An opening tag - <body>

   * The element’s “content”
     
   * A closing tag - </body> 
   
1. ___What is it called when you give an element extra information?___

   <ins>Attributes</ins> contain extra settings or information about the element. 
   
1. ***What is a semantic element?***
   
   A semantic element in HTML is an element that clearly describes its meaning and the type of content it contains, to both the browser and the developer. In HTML, the <ins>h1</ins> element is a semantic element.
